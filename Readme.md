# Stree Shakthi Landing Page

A responsive HTML landing page built from the provided Figma design using **HTML5**, **CSS3**, **Bootstrap 5**, and **Bootstrap Icons**.

---

## 📌 Project Overview

This project is a frontend implementation of the Stree Shakthi homepage. The objective was to convert the supplied Figma design into a responsive, reusable, and maintainable website while preserving the overall layout and visual hierarchy.

The page includes:

- Responsive Header/Navbar
- Hero Section
- About Section
- Vision & Mission
- Latest Highlights
- Statistics Counter
- Latest Publications
- Our Initiatives
- Partners/Organizations
- Community & Gallery
- Orders & Notifications
- Latest News & Events
- Testimonials
- Mobile App Promotion
- Footer

---

# Why These Technologies?

### HTML5

Used for creating a semantic and accessible page structure.

### Bootstrap 5

Bootstrap was selected because it provides:

- Responsive Grid System
- Mobile-first layout
- Faster development
- Utility classes
- Consistent spacing
- Responsive components

### CSS3

Custom CSS is used to achieve the exact Figma design since Bootstrap utilities alone cannot reproduce all the visual details such as:

- Custom gradients
- Rounded cards
- Shadows
- Hero overlays
- Decorative backgrounds
- Custom spacing
- Hover animations

### Bootstrap Icons

Used for:

- Search icon
- Navigation arrows
- Download icons
- Social media icons
- Accessibility icon

# Assumptions Made

Since only the Figma design was provided, the following assumptions were made:

### 1. Images

Assets/Images downloaded from Figma.

---

### 2. Static Content

All sections currently use static data.

---

### 3. Carousels

The Figma displays sliders for:

- Hero cards
- Latest Highlights
- Publications
- Initiatives
- Testimonials
- News

### 4. Search

The search field is present for UI purposes only.

Backend search functionality is outside the scope of this implementation.

---

### 5. Navigation Links

Navigation items currently point to each sections or '#'.

Actual routing can be added later.

---

### 6. Forms

Newsletter and search forms currently have no backend integration.

---

### 7. Icons

Bootstrap Icons were used.

---

# Responsiveness

The layout has been optimized for:

- Desktop (≥1200px)
- Laptop (992px–1199px)
- Tablet (768px–991px)
- Mobile (<768px)

Bootstrap's responsive grid system is used throughout the project.

---

# Performance Considerations

- Bootstrap loaded via CDN
- Optimized layout structure
- Reusable CSS classes
- Minimal DOM nesting
- Responsive images using `img-fluid`
- Lightweight custom CSS
